🎯 Lezione 1: Fondamenti JavaScript

Cos'è JavaScript?

JavaScript è il linguaggio di programmazione del web. Ti permette di rendere le pagine web interattive e dinamiche.

// Variabili e tipi di dati let nome = "Mario"; const eta = 25; var attivo = true; // Output nella console console.log("Ciao " + nome); console.log("Hai " + eta + " anni"); console.log("Attivo:", attivo);

Tipi di Dati Principali

JavaScript ha diversi tipi di dati: string (testo), number (numeri), boolean (vero/falso), array (liste), object (oggetti).

⚡ Lezione 2: Funzioni JavaScript

Cosa sono le Funzioni?

Le funzioni sono blocchi di codice riutilizzabili che eseguono un compito specifico. Puoi chiamarle quando ne hai bisogno.

// Dichiarazione di funzione function saluta(nome) { return "Ciao " + nome + "!"; } // Chiamata della funzione let messaggio = saluta("Anna"); console.log(messaggio); // Funzione con più parametri function somma(a, b) { return a + b; } console.log(somma(5, 3)); // Output: 8

🔄 Lezione 4: Cicli e Iterazioni

I Cicli in JavaScript

I cicli permettono di ripetere codice più volte. Sono essenziali per elaborare liste di dati o ripetere azioni.

// Ciclo for for (let i = 0; i < 5; i++) { console.log("Numero:", i); } // Ciclo while let contatore = 0; while (contatore < 3) { console.log("Contatore:", contatore); contatore++; } // Ciclo for...of per array let frutti = ["mela", "banana", "arancia"]; for (let frutto of frutti) { console.log("Frutto:", frutto); }

📊 Lezione 5: Array e Liste

Lavorare con gli Array

Gli array sono liste di elementi. JavaScript offre molti metodi per manipolarli: push, pop, map, filter, reduce.

// Creare e manipolare array let numeri = [1, 2, 3, 4, 5]; let nomi = ["Alice", "Bob", "Charlie"]; // Metodi principali numeri.push(6); // Aggiunge alla fine numeri.pop(); // Rimuove dall'ultimo console.log(numeri.length); // Lunghezza // Metodi avanzati let doppi = numeri.map(n => n * 2); let pari = numeri.filter(n => n % 2 === 0); let somma = numeri.reduce((acc, n) => acc + n, 0); console.log("Doppi:", doppi); console.log("Pari:", pari); console.log("Somma:", somma);

🏗️ Lezione 6: Oggetti JavaScript

Programmazione ad Oggetti

Gli oggetti sono collezioni di proprietà e metodi. Rappresentano entità del mondo reale nel tuo codice.

// Creare un oggetto let persona = { nome: "Mario", eta: 30, citta: "Roma", saluta: function() { return `Ciao, sono ${this.nome}`; } }; console.log(persona.nome); console.log(persona.saluta()); // Costruttore di oggetti function Auto(marca, modello, anno) { this.marca = marca; this.modello = modello; this.anno = anno; this.descrivi = function() { return `${this.marca} ${this.modello} del ${this.anno}`; }; } let miaAuto = new Auto("Toyota", "Corolla", 2020); console.log(miaAuto.descrivi());

⚡ Lezione 9: Programmazione Asincrona

Async, Await e Promises

La programmazione asincrona permette di gestire operazioni che richiedono tempo senza bloccare il codice.

// Promise function fetchData() { return new Promise((resolve, reject) => { setTimeout(() => { resolve("Dati ricevuti!"); }, 2000); }); } // Async/Await async function getData() { try { console.log("Caricamento..."); let result = await fetchData(); console.log(result); } catch (error) { console.error("Errore:", error); } } // Fetch API async function loadUser(id) { let response = await fetch(`/api/user/${id}`); let user = await response.json(); return user; }
