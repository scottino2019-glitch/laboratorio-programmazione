📝 Lezione 1: Fondamenti HTML

Cosa è HTML?

HTML (HyperText Markup Language) è il linguaggio di markup standard per creare pagine web. Utilizza "tag" per strutturare il contenuto.

<!DOCTYPE html> <html> <head> <title>La mia prima pagina</title> </head> <body> <h1>Benvenuto!</h1> <p>Questo è un paragrafo.</p> </body> </html>

🎨 Lezione 2: Styling con CSS

Introduzione al CSS

CSS (Cascading Style Sheets) controlla l'aspetto visuale degli elementi HTML. Puoi cambiare colori, dimensioni, posizioni e molto altro.

<style> h1 { color: blue; font-size: 2rem; text-align: center; } p { color: #333; font-family: Arial, sans-serif; line-height: 1.6; } </style>

📐 Lezione 3: Layout e Posizionamento

Flexbox per Layout Moderni

Flexbox è un potente sistema di layout che ti permette di allineare e distribuire elementi facilmente.

.container { display: flex; justify-content: center; align-items: center; gap: 20px; } .box { background: #667eea; color: white; padding: 20px; border-radius: 8px; }

📱 Lezione 4: Design Responsive

Media Queries per Dispositivi Mobili

Le media queries permettono di applicare stili diversi in base alla dimensione dello schermo.

/* Stili per desktop */ .container { display: flex; gap: 20px; } /* Stili per mobile */ @media (max-width: 768px) { .container { flex-direction: column; gap: 10px; } }

✨ Lezione 5: Animazioni CSS

Transizioni e Animazioni

Le animazioni CSS rendono le pagine web più dinamiche e coinvolgenti. Puoi animare colori, dimensioni, posizioni e molto altro.

/* Transizione semplice */ .button { background: blue; transition: all 0.3s ease; } .button:hover { background: red; transform: scale(1.1); } /* Animazione con keyframes */ @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-20px); } } .bouncing { animation: bounce 2s infinite; }
